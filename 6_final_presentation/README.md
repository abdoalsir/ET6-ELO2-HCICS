## **Key Highlights**

- **Scale:** 181 localities, **6.55M IDPs** analyzed
- **Validation:**
  - ML accuracy: **94.6%**
  - Moran’s I: **0.4918**, *p* < 0.001
- **Impact:** Identified **156 priority localities** needing mobile health
clinics
- **Deliverable:** Open-source, interactive dashboard for humanitarian
decision-making
